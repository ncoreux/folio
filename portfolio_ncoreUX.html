DataEZ
Data Management Framework, Web

The DataEz framework is highly modular, scalable, HITRUST certified and API driven, where Data ingested in the platform is tested for quality based on customized rules, classified for PHI/PII and tokenized or anonymized. Ingested data is cataloged and its data lineage captured for search and discovery.

Data scientists/data engineers can automate the entire process from ingestion to analysis in a single CLICK

This can plug in to various tools like Airflow, Keycloak, Tableau that may be required for data analysis
